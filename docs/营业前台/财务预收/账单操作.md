# 账单操作
::: tip
财务部的账单其实都是预收账单，这里面包括新办会员的账单、续充会员的账单、微商城售票的账单、票券的账单等。

进入账单之后还支持一系列的操作！
:::

## 二、账单操作
关于账单的一系列操作和**水会-账单**的操作完全一致，可以直接参考[水会-账单](../桑拿水会/账单操作.md)的相关操作。
