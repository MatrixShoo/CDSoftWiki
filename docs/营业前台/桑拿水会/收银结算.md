## 一、收银结账
### 1.普通付款结账
普通付款包括**现金、银联、支票**等付款方式，结算比较简单。
1. 右侧付款方式选择**现金、银联、支票**
2. 然后输入客人要支付的金额，按**回车**确认
3. 按**F9**或者点击结账按钮即可完成结账


![20240307172245](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240307172245.png)

![20240312102955](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240312102955.png)
:::

### 2.会员付款结账
#### 2.1 刷卡结账
   进入结账界面后，点击**VIP 会员（右Alt）**的图标区域，或者直接按键盘右边的**Alt**键即可调出**会员查询界面**
   ![20240312104855](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240312104855.png)
1. 系统默认已经勾选了**芯片F1**的选项，调出会员界面后直接刷卡即可，刷卡后会自动查询会员。
2. 查询出会员后，按**回车**或者点击**确认**即可
   ![20240312105344](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240312105344.png)
3. 这样就完成了会员刷卡，**扣款**金额会显示出来
   ![20240312105850](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240312105850.png)
4. 系统还支持多张会员卡一起结账，如果一张会员的金额不足以支付消费，那么在完成上面的操作后，接着再刷下一张会员卡即可
5. 最后别忘了点击**结账**

#### 2.2 卡号结账
::: warning
会员卡号结账相对于刷卡结账来说对客人造成资金的损失的风险更大，如果客人的会员卡没有设置密码的话，知道客人的卡号就可以使用客人的余额了，请门店谨慎选择开启，建议使用权限控制只允许主管经理以上才能使用！！！
:::
   进入结账界面后，点击**VIP 会员（右Alt）**的图标区域，或者直接按键盘右边的**Alt**键即可调出**会员查询界面**
   ![20240312104855](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240312104855.png)
1. 因为系统默认已经勾选了**芯片F1**的选项，所以你需要选择**卡号F5**选项，可以鼠标点击，也可以按快捷键**F5**，然后输入**卡号**
2. 因为是手动输入，系统不会自动回车查询，你需要按**回车**键或者点击**查询**按钮完成**查询**会员操作，查询出会员后，按**回车**或者点击**确认**即可
   ![20240312110712](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240312110712.png)
3. 这样就完成了会员卡号结账，**扣款**金额会显示出来
   ![20240312105850](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240312105850.png)
4. 系统还支持多张会员卡一起结账，如果一张会员的金额不足以支付消费，那么在完成上面的操作后，接着再输入查询下一张会员卡号即可
5. 最后别忘了点击**结账**

5. 最后别忘了点击**结账**

#### 2.4 条码结账
::: tip
会员条码结账的前提是客人的实体会员卡已经绑定到了微信会员上，或者本身办理的就是电子会员卡，如果实体卡还没有绑定，需要先帮助他绑定到微信会员上才可以进行会员条码结账！！！
:::
进入结账界面后，点击**VIP 会员（右Alt）**的图标区域，或者直接按键盘右边的**Alt**键即可调出**会员查询界面**
   ![20240312104855](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240312104855.png)
1. 因为系统默认已经勾选了**芯片F1**的选项，所以你需要选择**条码F12**选项，可以鼠标点击，也可以按快捷键**F12**，然后扫描**电子会员码**，扫描后系统会自动查询出会员
2. 查询出会员后，按**回车**或者点击**确认**即可
   ![20240312112415](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240312112415.png)

5. 最后别忘了点击**结账**

#### 2.4 短信结账
::: tip
短信结账就是先给客人的手机发送一条验证码，客人告诉收银员这个验证码才可以结账，相对于其他的结账方式是比较方便的。
:::
进入结账界面后，点击**VIP 会员（右Alt）**的图标区域，或者直接按键盘右边的**Alt**键即可调出**会员查询界面**
   ![20240312104855](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240312104855.png)
1. 进入会员界面后，点击**短信F4**按钮或者按快捷键**F4**，打开**短信验证**界面
2. 输入客人**会员卡预留的手机号码**，然后按**回车**或者点击**获得验证码**按钮来触发验证码发送
3. 在**验证码**输入框内填入客人告知的**验证码**，然后按**回车**或者点击**确认**按钮即可查询出会员卡
   ![20240312113414](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240312113414.png)
4. 查询出会员后，按**回车**或者点击**确认**，这样就完成了会员卡号结账，**扣款**金额会显示出来
   ![20240312105850](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240312105850.png)
5. 系统还支持多张会员卡一起结账，如果一张会员的金额不足以支付消费，那么在完成上面的操作后，接着再使用手机验证码查询下一张会员卡号即可
6. 最后别忘了点击**结账**

### 3.票券付款结账
#### 3.1 抖音美团券
::: tip
系统已经和抖音、美团做好了核销接口，客人购买的抖音、美团券可以直接在系统内扫码结账核销，而无需使用他们提供的核销工具，这样极大的方便收银的工作。
:::


::: warning
1. **票券**框内的**券号F4**可以手动输入票券号，在扫描平台损坏的情况下可以手动输入客人的票券号。
2. 受制于抖音券核销接口的限制，抖音券的券号不可以在系统内手动输入，只可以使用扫描平台扫描。
3. 美团券的券号可以手动输入来核销使用。
:::


1. 进入结账界面后，**不需要点击任何按钮，直接在扫描平台上扫描票券码**，系统即可自动识别票券。如果有多张，可以连续扫描。
2. 识别出来的票券会在**票券**框内显示
   ![20240312121832](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240312121832.png)
3. 如果消费已经全部抵扣完了，最后点击**结账**即可
   
#### 3.2 微商城票券

::: warning
1. **票券**框内的**券号F4**可以手动输入票券号，在扫描平台损坏的情况下可以手动输入客人的票券号。
2. 微商城的券号可以手动输入来核销使用。
:::


1. 进入结账界面后，**不需要点击任何按钮，直接在扫描平台上扫描票券码**，系统即可自动识别票券。如果有多张，可以连续扫描。
2. 识别出来的票券会在**票券**框内显示
   ![20240312122739](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240312122739.png)
3. 如果消费已经全部抵扣完了，最后点击**结账**即可

#### 3.3 输入票券票数
::: tip
输入票数结账一般的使用场景比如：发放的纸质券，但是票号不是系统生成的；或者没有使用系统的抖音美团的接口来核销票券，而是使用官方提供的工具来核销的情况
:::

   进入结账界面后，点击**票券（左Alt）**的图标区域，或者直接按键盘左边的**Alt**键即可调出**选择票种界面**
   ![20240312124611](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240312124611.png)
1. 在左侧票种出你能看到目前你们可用的票种，如果太多不好找的话，可以在上方助记码处输入票券名称的首字母来搜索
   ![20240312123416](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240312123416.png)
2. 选择客人要使用的票种，在对话框中输入要使用的张数（默认1张），然后按**回车**或者点击**确认**，如果还有需要使用的，那就重复这一步骤，选择好的票会在右侧**核销票种**处显示，这一步我们称为**选择票种**。
   ![20240312123613](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240312123613.png)

   ![20240312123814](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240312123814.png)
3. 刚刚选择好了票种，都在右侧显示了，这时候我们就可以点击下方的**确认F5**把票种提交到结账付款方式了。提交的票券会在**票券**框内显示
   ![20240312124322](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240312124322.png)
4. 如果消费已经全部抵扣完了，最后点击**结账**即可
### 4.付呗付款结账

::: tip
系统已经集成了微信支付宝的扫码扣款接口，使用付呗第三方支付可以无需关心客人使用的是微信还是支付宝，直接扫码扣款即可，提高收银效率
:::

1. 进入结账界面后，点击右侧**付款**内的**付呗（或者按付呗旁边的快捷键）**
   ![20240312125411](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240312125411.png)
2. 输入**本次支付**金额，一般默认即可，按**回车**或者点击**确认**按钮
   ![20240312125254](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240312125254.png)
3. 按**F9**结账或者点击**结账**按钮
4. 弹出的**扫码付款**，扫描客人的**付款码**即可，无需关心客人使用的是微信还是支付宝，通通可以直接扫。
   ![20240312125325](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240312125325.png)
5. 扫码后会有**扫码进程**，如果需要客人**输入密码**会有文字显示，请提醒客人。

3. 按**F9**结账或者点击**结账**按钮

   ![20240312130227](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240312130227.png)
5. 最后还会弹出**授权**，需要有相关权限的人员来授权**挂账**的操作

   ![20240312130708](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240312130708.png)

## 二、折扣相关
### 1.整单折扣
::: tip
整单折扣的前提是需要先添加设置好折扣方案，然后结账的时候使用这个折扣方案！
:::
比如现在有两个手牌结账，我们需要将将他们原价98的门票都打折成78，下面是操作步骤：
![20240312152007](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240312152007.png)
1. 正常刷手牌进入结账界面
2. 点击下方的**折扣方案**按钮，选择你需要使用的**折扣方案**，点击**确认**
   ![20240313103807](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240313103807.png)
3. 这样我们就完成了**整单折扣**的操作，实收已经变为78了，下面正常付款结账即可。
   ![20240313103937](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240313103937.png)

### 2.单项折扣
::: tip
**单项折扣**指的是：可以修改一个或者多个**项目**的**折扣率**或者**实收单价**，其实就是针对这些项目做打折处理！
:::

比如6003和6005都消费了门票，原价都是98，因为一些原因，我们需要把6005的门票打折到50，下面是操作步骤：
![20240313105012](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240313105012.png)

1. 正常刷手牌进入结账界面
2. 点击下方的**单项折扣**按钮，系统会弹出**单项折扣**的修改框，这个时候我们只需要修改6005手牌的门票的**实收**，然后确认即可
      ::: info
      如果是需要给门票打8折，那么就修改**折扣率**为80
      :::
   ![20240313105149](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240313105149.png)
3. 这样我们就完成了**单项折扣**的操作，实收已经变为50了，下面正常付款结账即可。
   ![20240313105600](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240313105600.png)

### 3.类别折扣
::: tip
**类别折扣**指的是：可以修改一个或者多个**项目类别**的**折扣率**或者**实收单价**，其实就是给属于这个**项目类别**下的**所有消费项目**做打折处理！
:::
比如6001手牌消费了门票和自助餐，那么消费项目类别就是**门票类**和**自助餐类**，因为一些原因我们需要给这个手牌的所有自助餐打8折，下面是操作步骤：
![20240313105944](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240313105944.png)


1. 正常刷手牌进入结账界面
2. 点击下方的**类别折扣**按钮，系统会弹出**类别折扣**的修改框，这个时候我们只需要修改需要打折的类别的**折扣率**，8折就写80，然后确认即可
   ![20240313110320](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240313110320.png)
3. 这样我们就完成了**类别折扣**的操作，自助餐项目的实收已经打了8折了，下面正常付款结账即可。
   ![20240313110401](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240313110401.png)


## 三、其他功能
### 1.账单预览
**账单预览**就是打印一张**预结单**给客人看自己的消费的，如果收银台没有**双屏账单**显示，这个功能比较实用！
1. 正常刷手牌进入结账界面
2. 按键盘快捷键**F5**或者点击左下方**预览F5**按钮即可打印一张。
   ![20240313111141](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240313111141.png)

### 2.消费录入
::: tip
**收银界面**的**消费录入**功能主要就是方便收银员可以快速跳转到录入界面，而**不需要退出**后再进去录入界面
:::

1. 正常刷手牌进入结账界面
2. 点击下方的**消费录入**按钮，如果结账手牌有多个，会让你选择需要**录入**的手牌，如果结账手牌只有1个，就会直接进入**录入界面**
   ![20240313111613](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240313111613.png)
3. 进入录入界面后就可以正常进行**消费录入**了，关于如何录入可以参考：[如何操作入单](../桑拿水会/点单录入#三、操作入单)

### 3.添加手牌
::: tip
就是手动添加要结账的手牌，如果是刷手牌的话可以直接刷手牌，不需要操作这个功能！
:::
1. 正常刷手牌进入结账界面
2. 点击下方的**添加手牌**按钮，然后在对话框**厅牌号**框内输入手牌号，输入一个回车一下，然后输入下一个再回车，最后**确认**即可
   ![20240313112555](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240313112555.png)
3. 这样就添加好结账手牌了。

### 4.移除手牌
::: tip
和上面添加手牌相反的操作效果
:::
1. 正常刷手牌进入结账界面
2. 点击下方的**移除手牌**按钮，然后在对话框中点击需要移除的手牌的右侧的**移除**按钮，最后**确认**即可
   ![20240313112914](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240313112914.png)
3. 这样就移除掉结账手牌了。

### 5.保留手牌
::: tip
**保留手牌**就是只将手牌的内消费结账，但是手牌**不离场**，还要继续在场消费，手牌的状态还是**有人**，我们又称之为**留牌结账**
:::
比如现在有6001和6002两个手牌需要将手牌内的消费都结账，但是手牌不离场，我们就可以操作**保留手牌**：
1. 正常刷手牌进入结账界面
2. 点击下方的**保留手牌**按钮，然后在对话框中勾选需要保留的手牌，最后**确认**即可
    ::: tip
   如果需要操作的手牌号比较多，你也可以点击**确认**旁边的**全部留牌**和**取消留牌**
   :::  
   ![20240313113738](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240313113738.png)
3. 这样操作后这个手牌结账后就不会离场，然后结账还是正常操作流程！
   ![20240313114448](https://wiki-cdsoft.oss-cn-hangzhou.aliyuncs.com/20240313114448.png)
